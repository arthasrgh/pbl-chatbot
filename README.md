# PROJECT PBL - Dashboard Chatbot WhatsApp Bangkesbangpol

## Deskripsi
Project ini merupakan sistem dashboard chatbot WhatsApp berbasis:

- Frontend : Vue.js + Vite
- Backend API : Laravel
- Bot WhatsApp : whatsapp-web.js + Node.js
- Database : MySQL

Fitur utama:
- Login Admin
- Dashboard Statistik
- Riwayat Chat User
- Manajemen User
- Grafik Chat Harian
- Bot WhatsApp Otomatis
